# QA-manual-testing-saucedemo
# 🧪 SauceDemo Manual Testing Project

## 📌 Project Overview

This project demonstrates manual testing of an e-commerce web application using the SauceDemo platform. The goal is to validate core functionalities such as login, product selection, cart management, and checkout process.

## 🌐 Application Under Test

* Website: SauceDemo
* Type: E-commerce (Demo Testing Site)

## 🎯 Objective

* Perform manual testing on key features
* Identify bugs and usability issues
* Document test cases and results

## 🔍 Scope of Testing

* Login functionality
* Product listing page
* Add to cart / Remove from cart
* Cart validation
* Checkout process

## 🧰 Testing Types Used

* Functional Testing
* UI Testing
* Negative Testing

## 📊 Test Artifacts Included

* ✅ Test Plan
* ✅ Test Cases (Excel)
* ✅ Bug Report
* ✅ Test Summary Report

## 🐞 Bug Reporting

Bugs were identified and documented with:

* Steps to reproduce
* Expected vs Actual result
* Severity level

## 📈 Test Summary

* Total Test Cases: 30
* Passed: 25
* Failed: 5
* Bugs Found: 5
